sklearndf.classification.StackingClassifierDF¶

class sklearndf.classification.StackingClassifierDF(*args, **kwargs)[source]¶

Stack of estimators with a final classifier.

Note

This class is a wrapper around class sklearn.ensemble.StackingClassifier. It provides enhanced support for pandas data frames, and otherwise replicates all parameters and behaviours of class StackingClassifier.

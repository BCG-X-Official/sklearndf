sklearndf.clustering.BirchDF#

class sklearndf.clustering.BirchDF(*, threshold=0.5, branching_factor=50, n_clusters=3, compute_labels=True, copy=True)[source]#

Implements the BIRCH clustering algorithm.

Note

This class is a wrapper around class sklearn.cluster.Birch. It provides enhanced support for pandas data frames, and otherwise delegates all attribute access and method calls to an associated Birch instance.

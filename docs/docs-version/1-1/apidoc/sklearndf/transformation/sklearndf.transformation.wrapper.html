sklearndf.transformation.wrapper¶

Wrapper classes to enhance the functionality of native transformers conforming with the scikit-learn API.

Classes¶

AdditiveChi2SamplerWrapperDF

DF wrapper for sklearn.kernel_approximation.AdditiveChi2Sampler.

BaseDimensionalityReductionWrapperDF

Base class of DF wrappers for dimensionality-reducing transformers.

BaseMultipleInputsPerOutputTransformerWrapperDF

DF wrapper for transformers mapping multiple input columns to individual output columns.

ColumnPreservingTransformerWrapperDF

DF wrapper for transformers whose output columns match the input columns.

ColumnSubsetTransformerWrapperDF

Abstract base class of DF wrappers for transformers that do not change column names, but that may remove one or more columns.

ColumnTransformerWrapperDF

DF wrapper for sklearn.compose.ColumnTransformer.

ComponentsDimensionalityReductionWrapperDF

Base class of DF wrappers for dimensionality-reducing transformers supporting the components_ attribute.

FeatureSelectionWrapperDF

DF wrapper for feature selection transformers.

ImputerWrapperDF

DF wrapper for imputation transformers, e.g., sklearn.impute.SimpleImputer.

IsomapWrapperDF

DF wrapper for sklearn.manifold.Isomap.

MissingIndicatorWrapperDF

DF wrapper for sklearn.impute.MissingIndicator.

NComponentsDimensionalityReductionWrapperDF

Base class of DF wrappers for dimensionality-reducing transformers supporting the n_components attribute.

NumpyTransformerWrapperDF

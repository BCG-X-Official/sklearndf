sklearndf.regression.BaggingRegressorDF¶

class sklearndf.regression.BaggingRegressorDF(*args, **kwargs: Any)¶

A Bagging regressor.

Note

This class is a wrapper around class sklearn.ensemble.BaggingRegressor. It provides enhanced support for pandas data frames, and otherwise replicates all parameters and behaviours of class BaggingRegressor.

sklearndf.wrapper.MetaEstimatorWrapperDF¶

class sklearndf.wrapper.MetaEstimatorWrapperDF(*args, **kwargs)[source]¶

Abstract base class of wrappers for estimators implementing sklearn.base.MetaEstimatorMixin.

A meta-estimator will call the methods of the embedded estimator using a modified copy of the X and y parameters, so we need to make sure that these are converted back to data frames.

This class covers three variants used in sklearn:

  • one delegate estimator in attribute estimator

  • one delegate estimator in attribute base_estimator

  • multiple delegate estimators in attribute estimators

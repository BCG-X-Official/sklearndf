sklearndf.classification.VotingClassifierDF¶

class sklearndf.classification.VotingClassifierDF(*args, **kwargs: Any)¶

Soft Voting/Majority Rule classifier for unfitted estimators.

Note

This class is a wrapper around class sklearn.ensemble.VotingClassifier. It provides enhanced support for pandas data frames, and otherwise replicates all parameters and behaviours of class VotingClassifier.

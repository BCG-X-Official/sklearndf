sklearndf.regression.GaussianProcessRegressorDF¶

class sklearndf.regression.GaussianProcessRegressorDF(*args, **kwargs)[source]¶

Gaussian process regression (GPR).

Note

This class is a wrapper around class sklearn.gaussian_process.GaussianProcessRegressor. It provides enhanced support for pandas data frames, and otherwise replicates all parameters and behaviours of class GaussianProcessRegressor.

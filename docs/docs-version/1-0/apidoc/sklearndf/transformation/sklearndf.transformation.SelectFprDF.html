sklearndf.transformation.SelectFprDF¶

class sklearndf.transformation.SelectFprDF(*args, **kwargs)[source]¶

Filter: Select the pvalues below alpha based on a FPR test.

Note

This class is a wrapper around class sklearn.feature_selection.SelectFpr. It provides enhanced support for pandas data frames, and otherwise replicates all parameters and behaviours of class SelectFpr.

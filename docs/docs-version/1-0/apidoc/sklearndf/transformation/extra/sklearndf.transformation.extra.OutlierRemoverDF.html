sklearndf.transformation.extra.OutlierRemoverDF¶

class sklearndf.transformation.extra.OutlierRemoverDF(iqr_multiple=3.0)[source]¶

Remove outliers according to Tukey’s method.

A sample is considered an outlier if it is outside the range \([Q_1 - iqr\_ multiple(Q_3-Q_1), Q_3 + iqr\_ multiple(Q_3-Q_1)]\) where \(Q_1\) and \(Q_3\) are the lower and upper quartiles.

Bases

BaseEstimator, TransformerDF

Metaclasses

ABCMeta

Parameters

iqr_multiple (float) – the multiple used to define the range of non-outlier samples in the above explanation (defaults to 3.0 as per Tukey’s definition of far outliers)
